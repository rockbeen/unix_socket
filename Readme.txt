На языке программирования Си написать клиент-серверное приложение, использующее UNIX сокеты;
Функционал клиента: передача сообщения серверу.
Функционал сервера:
    • сервер при запуске в режиме сервиса (daemon) создает средство взаимодействия;
    • после получения кодового сообщения сервер уничтожает средство взаимодействия и завершает работу;
    • сервер ведет журнал, в котором фиксируется время запуска и завершения сервера, а также сообщения от клиентов с указанием времени их получения.
Для работы с файлами использовать системные вызовы UNIX.
